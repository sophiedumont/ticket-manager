# TICKET MANAGER




## Installation & Set up

```bash
$ docker-compose up -d
```

On first start. Launch fixtures with

```bash
$ docker-compose exec backend npm run fixtures
```

## Accessing the app

```
# Back-end
http://localhost:3000

# Back-end documentation
http://localhost:3000/api

# Front-end
$ http://localhost:3001

# Admin
$ http://localhost:3002
```
